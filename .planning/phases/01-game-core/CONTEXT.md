# Phase 01: Game Core

## Tech Decision
- HTML5 Canvas + JavaScript (чистый браузер, без сервера)
- Почему: максимальная доступность, не требует установки, работает по ссылке

## Design
- Темно-зелёный лабиринт из грядок
- Оранжевые морковки на грядках
- Белые зайцы с анимацией прыжков
- Зелёный питон (змейка) с анимацией движения
- Пиксель-арт стиль или минималистичный
