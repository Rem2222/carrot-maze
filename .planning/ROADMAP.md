# ROADMAP: Carrot Maze Game

## Goal
Браузерная игра: питон охотится на зайцев в лабиринте из грядок с морковкой.

## MVP (Playable)
- Лабиринт из грядок с морковкой
- Зайцы, прыгающие между грядок
- Питон, управляемый игроком (стрелки/WASD), охотится на зайцев
- Счётчик очков
- Рестарт игры

## Phases
- 01-game-core: Игровое поле, питон, зайцы, базовый геймплей
